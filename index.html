换一个房子，换一种生活

发表于 2020-03-04

换一个房子，换一种生活

标签（空格分隔）： 创作

长沙这座城市，没有想象中来的那么温暖，特别是对于一个外来人。

大学毕业之后，来到这座城市工作，刚开始的工作包吃包住，每个月拿着不高的薪水，却也还活的舒服。在第一家公司度过了一年之后，觉得自己没有任何进步，想要打破这种状态。于是和女朋友一起裸辞。从长沙的郊区来到了长沙的市区。

裸辞一时爽，生活泪两行。出来之后，面临的第一个问题，就是住在哪里。幸好有一个朋友的公寓，还没有出租出去，有一个月的空闲期。我就暂时借助在这个朋友的公寓里。刚开始面对新生活，一切都是美好的，自己拿着公司发的400元购物卡，去买了锅碗瓢盆（生活就是这么现实），去超市买了自己喜欢吃的菜。至今我还记得，第一顿吃的是火锅。自己卖菜做饭，自己洗碗拖地，乐在其中。

还没有在自己感觉很小资的生活体验几天，找工作和找房子的问题就来了，由于临近过年，工作不是很好找，于是想着先把房子找了，以后就在房子附近找工作。在网上平台看了几个比较满意了，找到中介去看房，看完房子问价格，完全不是网上的那个价格(中介解释说是因为年后就是租房高峰了，所以房租涨价了），再加上需要一个月的中介费，摸摸自己的口袋，灰溜溜的从中介那里逃离开了。

很快就过年了，自己过年没有回家，因为自己本身就没什么钱，不好意思回去，更多的其实还是车票太贵了，自己舍不得花。女朋友住的比较近，就回家过年去了。于是我一个人在朋友的房子里，度过了这个春节。除夕那天，自己拿出早就买好的火锅底料，放上肉、蔬菜和丸子，就组成了丰盛的年夜饭，这个时候的火锅，远远没有了刚开始的那种新鲜感和美味，更多的是为了生活填饱肚子。自己边吃饭边想，今年一定要好好奋斗，攒一点钱，过上更好的生活。正吃着的功夫，父亲打来电话，问候我一个人的年过的怎么样？看着火锅里没吃多少的菜，我告诉父亲，外派过的很好，不需要担心我。

年后就是繁忙的找工作，面试了几家公司，感觉都不是很好。这一次找工作，自己精挑细选，想找一个不将就，能够长期做下去的工作。这个时候就特别厌恶那些发布虚假信息的骗子公司，招聘条件、待遇写的那么好，去了完全不是那么回事，我就想不通，做人诚实一点就不好吗？这样大家都节省时间，还能留个好印象。

记不清楚是什么时候面试的一家公司，在我准备坐地铁回家的时候发来一条消息，工资4000，职位总经理助理，下周一可以入职。我当时心想，是哪个总经理这么廉价，他的助理才值这么点钱？随即又收到朋友发来的消息：”兄弟，房子还需要住多久？有个租户想租下来”。看到这里，我内心比较复杂，什么时候自己也变成了那种被人要赶出去的人？自己毕业这一年到底做了什么，怎么混的这么差？于是咬咬牙，拿起手机回复了那个公司HR发来的消息，可不可以加500元钱？我是真很喜欢这份工作，希望能够长久做下去。如果你能加点钱，我一定不会让你失望的。HR回复，我去请示下，稍后回复你。

晚上，我收到了HR发来的消息，公司同意给你加500元钱，你下周一能来上班吗？此刻的我，只要有个工作，确定有经济来源就好了，我就可以马上去公司附近租个房子，给朋友把房子腾出来了。我叹了口气，回复了：可以，周一见。

第二天，我就在跑到公司附近的小区，去看有没有要出租的房子。一则918元的房子吸引了我，我拨打了房东的电话，很快就上楼去看了房子。房子便宜不是没有道理的，一间放下床之后，就没什么空间的房子，但是有一个阳台，阳台上改造了一个厨房，还有一个独立的卫生间。小是小了点，但是能做饭，能睡觉，还便宜。交通也还方便，电梯上下楼。想着自己的经济情况，当下就决定租下来了。房东随身带着合同，马上签字付款。压一付三，交完房租之后，自己身上还剩下200元钱。就是接下来一个月的生活费了。时间不等人，马上回到公寓，收拾好东西，下午就从朋友那里搬了出来。给朋友发了条消息，我找到新的地方住了，下午已经搬走了，房间也整理好了，谢谢你这段时间能帮房子借给我住，下个月我发工资了请你吃饭。

从离职，到现在，过去20天了，终于有了一个属于自己的地方，要好好开始奋斗了。

没有任何路是容易的，坚持才会胜利

发表于 2020-03-04

#没有任何路是容易的，坚持才会胜利

标签（空格分隔）： 随笔

    新一年的开始不是那么顺利，在家吃完了26天的药，脸上应该是由于中药药性的原因，长了很痘痘。 脸上变得满目疮痍。幸好最近由于新冠状病毒的爆发，出门需要带口罩，可以遮掩下，不然的还是比较吓人的。

    以前总觉得，时间不够用，现在时间充裕了，自己需要抓住。首先这段时间，要把写日记这件事坚持下来，然后每天需要锻炼身体，出汗洗澡。目前每天还是能早睡早起的，要比之前好一点，但是还不够。想要获得成功，就一定要坚持。

    个人觉得目前除了工作之外，想要实现财务自由，就是选定一条路子，扎根进去做。现在对于我而言，能够选择的，可能就是文字、淘宝以及钻研技术了。当然了，现在时间充裕，工作也需要做好。明天把业务相关的行业，企业等梳理下。然后按照计划去拜访或者电话联系。每个人迈出第一步都很难，现在这个机会对自己而言是真的很好，一定要抓住。

    打破困境最好的方法，就是先改变自己，变成一个优秀的自己。加油！

第一天，迈出舒适区

发表于 2020-02-19

第一天，迈出舒适区

标签（空格分隔）： 生活

    要改变自己的生活的习惯，往往第一天是很难做到的，毕竟之前那么多次都没能改变。但是今天一切变的出奇的顺利，昨天晚上，11点左右就睡着了，早上7点自然醒。本来起床之后，准备还接着躺在床上的。只要接着躺着，就肯定会睡着，所以我直接起来了。

    早起的感觉是很幸福的，因为我又很多的时间，可以做很多的事情。我本来想好好的洗个脸，但是却没有找到洗面奶，我找到钟阳的修眉刀，用他刮了胡子。我本俩还准备抹点护肤品的，但是都没有找到称心如意的。我开始变的径精致，早起之后要打理自己，这是对生活的态度，也是对人生的态度。时刻以最好的面貌展现在别人面前！

    第一天如此容易的完成早睡早起的目标，后面的每一天都会变得很困难，只需要一个念头，人就会放纵自己，使自己的努力白费，回到以前的样子。以前的生活看上去很舒适，但是是颓废的，看不到未来和希望的。起来之后，我来到客厅，打开电脑，审阅了昨天的订单，了解了商品的进价和售价，这种感觉太好了。

    既然早起了，光阴不应该被浪费，今天我白天就要开始学习淘宝视频了，今天的目标就是，将淘宝的店铺装修好。下午看下要不要出去钓鱼，晚上的话，就在知乎找一个问题，认认真真的写一篇回答（或者自己写一篇小说）；

加油，共勉！

今天起，好好生活

发表于 2020-02-18

#今天起，好好生活！

标签（空格分隔）： 日记

    疫情来临的时候，才知道自己活的是多么失败。面对没有住却还是继续要交的房租，面对没有上班没有收入还要还的车贷，面对继续生活所还需要的成本，新冠状病毒揭开了失败人生的遮羞布。

     自己无数次想要好好开始，追寻成为自由工作者的生活。目前的自己没有在任何一个行业深度耕耘，就是工作，也没有在一个公司待到两年以上。从现在开始，自己就好好开始自己的人生，并且将自己的奋斗记录下来，希望能在成功的时候，激励更多的人。

    目标很简单，就是发展一份自己的事业，开一个网上店铺；顺便成为已经小说家。当下要做到的事情有如下：

  • 1.每天11点前睡觉，早上6点起床
  • 2.不再游戏
  • 3.每天写一篇日记（也可以是文章），保证质量
  • 4.学习淘宝和亚马逊开店（看视频，选品，看相关的文章），在2月底之前，将淘宝的规则以及店铺开起来。
  • 5.每周看一本书。
  • 6.多加家务活，不要偷懒。
  • 7.每周写一个短篇故事，回答一个知乎问题。

以上，即为近期要做的事情。加油，共勉！

对不起，我没有成为你们的骄傲

发表于 2019-12-18

对不起，我没有成为你们的骄傲

​ 当我再一次向父母伸手要钱的时候，我才深深的意识到，对于生活，自己是这般无力。
小时候，想要买的东西，都需要向父母要钱，总是希望自己快点长大，觉得长大之后，自己就能做自己想做的事情，买自己想买的东西。
​ 于是拼了命的读书，考上一所好大学。一位毕业了，就会混的风声水起，可以到处去旅游，可以给父母尽一份孝心。未曾想到，毕业后的自己，做着身不由己的事情，拿着微不足道的薪水，很多次面对生活的艰辛，都需要像父母伸手。
​ 父母最大的愿望，就是我不要像他们一样从事体力劳动，可以天天坐在电脑前面，成为一个白领。而我，连这个基本的要求都没有达到。
